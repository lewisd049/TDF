Shlink CSV Upload Guide

Follow these steps to add the Shlink servers CSV file to app.shlink.io.

Step 1

Download the Shlink servers CSV file using the button below:

Step 2

Open the Shlink Web Client:

https://app.shlink.io

Step 3

In Shlink Web Client:

  1. Open the settings menu
  2. Find the server import option
  3. Select the downloaded CSV file
  4. Upload the file
  5. Your Shlink server should now appear

Important

Keep your CSV file private because it contains your Shlink server API keys.
